## 연습문제 3-2

### 문제

`x`가 0보다 큰 실수이고, `n`이 0보다 큰 양수일 때, `x`의 `n`승을 구하는 함수를 재귀로 구현해 보자.

### 코드

```kotlin
fun power(x: Double, n: Int): Double = when {
  n == 0 -> 1.0
  n == 1 -> x
  else -> x * power(x, n - 1)
}
```

### 풀이

- `x`의 0승은 1이므로, 1.0을 반환한다.
- `x`의 1승은 `x`이므로, `x`를 반환한다.
- `x`의 `n`승은 `x`의 `n-1`승과 `x`의 곱이므로, `x * power(x, n - 1)`을 반환한다.
